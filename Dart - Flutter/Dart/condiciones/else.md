Se usa en conjunto de condicionales como if para declarar que se ejecutara otra cosa si cierta condición no se cumple o si otra condición se cumple.

`init experienceYears = 5`

`if (experienceYears > 8) {`
	`print ("Eres un programador SENIOR")`
`} else if ( experienceYears > 5 ) {`
	`print("Eres un programador MID")`
`} else {`
	`print ()"Eres un programador JUNIOR")`
`};`
