Es una variable de texto siempre va entre comillas ' ' o " ".

String name = 'Mi nombre cualquier numero 32 y caracteres especiales'

Las String se pueden concatenar que es unir 2 variables.

String name = 'Yo'
String apellido = 'Soy'

String fullText  = name + apellido \\ Yo Soy
String altFullText = "$name y $apellido" \ En este caso el signo dolar manda a llamar a cualquier variable y la presenta como un string dando como resultado: "Yo y Soy"
