remove() permite retirar el primer argumenta que coincida con el argumento indicado

  names.remove("Aris");

===

removeAt

removeAt() permite retirar el argumento que ocupe el indice designado a la función.

  names.removeAt(1);

===

removeAll

removeAll() permite retirar todos los argumentos que coincidan con el argumente que se le brinde a la función

names.removeAll(names2);
