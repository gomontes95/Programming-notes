| Propiedad       | Qué hace                                 |
| --------------- | ---------------------------------------- |
| `Expanded`      | Toma todo el espacio libre en Row/Column |
| `Flexible`      | Similar a Expanded pero flexible         |
| `Spacer`        | Espacio vacío flexible                   |
| `SizedBox`      | Caja con tamaño fijo o espacio           |
| `AspectRatio`   | Mantiene proporción ancho/alto           |
| `FittedBox`     | Escala hijo para que quepa               |
| `Positioned`    | Posiciona en Stack                       |
| `Center`        | Centra                                   |
| `Align`         | Alineación personalizada                 |
| `Wrap`          | Flujo automático cuando no cabe en línea |
| `LayoutBuilder` | Da acceso a constraints del padre        |
|                 |                                          |
