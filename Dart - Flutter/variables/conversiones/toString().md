Es una función de variables que permite transformar una variable numerica a un string;

int numbersDemo = 615;
String stringsNumbers = numbersDemo.toString();

print(stringsNumbers) // hara el print de 615 como un strings

- toStringAsFixed()
Esta función nos permite dale un limitante a una variable numerica convertiendola en un numero. por ejemplo al hacer multiplicaciones con doubles o nums se pueden generar una cantidad enorme de decimales.
